# What is NumPy?

---

# Introduction

NumPy stands for:

```text
Numerical Python
```

NumPy is one of the most popular Python libraries for numerical and scientific computing.

It provides a powerful data structure called an:

```text
Array
```

which is faster and more memory-efficient than Python lists.

---

# Why Was NumPy Created?

Python lists are flexible but not optimized for heavy mathematical computations.

Example:

```python
numbers = [1, 2, 3, 4, 5]
```

For small data, lists work perfectly.

However, when working with:

* Thousands of values
* Millions of values
* Mathematical calculations
* Scientific computations
* Machine Learning datasets

Python lists become slower.

NumPy solves these problems.

---

# What Can NumPy Do?

NumPy can perform:

* Mathematical Operations
* Statistical Calculations
* Matrix Operations
* Linear Algebra
* Random Number Generation
* Data Manipulation
* Scientific Computing

# NumPy vs Python Lists

## Python List

```python
numbers = [1, 2, 3, 4]
```

---

## NumPy Array

```python
import numpy as np

numbers = np.array([1, 2, 3, 4])
```

---

# Advantages of NumPy Arrays

| Feature                  | Python List | NumPy Array |
| ------------------------ | ----------- | ----------- |
| Speed                    | Slow        | Fast        |
| Memory Usage             | Higher      | Lower       |
| Mathematical Operations  | Limited     | Powerful    |
| Machine Learning Ready   | No          | Yes         |
| Multidimensional Support | Basic       | Excellent   |
